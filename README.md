# Link-Locker

Link-Locker is a PHP + MySQL + Vanilla JavaScript web app for saving useful links, images, PDFs, screenshots, and notes as visual preview cards.

## Local setup

1. Copy this folder to:

```text
D:\Xampp\htdocs\link-locker-xampp
```

2. Start Apache and MySQL from XAMPP.

3. Open phpMyAdmin:

```text
http://localhost/phpmyadmin
```

4. Import:

```text
sql/schema.sql
```

5. Open the app:

```text
http://localhost/link-locker-xampp
```

## Database

Default database name: `link_locker`

Default local config is in:

```text
config/db.php
```

## Notes

Do not upload real user files from the `uploads` folder to GitHub.
